Проєкт призначений для аналізу цін між магазинами.

Нам поступає товар - див файл Input.xlsx

З ним треба порівнювати ціни у конкурентів - це два файли:
agrodoctor_prices.xlsx
tvk_ramos_prices.xlsx

Утиліта яка порівнює ціни - check_prices_urls.py

Вона для кожного товару шукає ціни в обох конкурентів і записує результат у новий
файл Input_analyzed_with_urls_YYYY-MM-DD.xlsx (до назви дописується дата аналізу,
наприклад Input_analyzed_with_urls_2026-07-21.xlsx).

Запуск:
    pip install -r requirements.txt
    python check_prices_urls.py

Логіка за якою працює утиліта:

1. Перевіряєм для кожного рядка з файла Input.xlsx за значенням з колонки "Артикул" співпадіння в таблицях з цінами конкурентів.
   Важливі два моменти:
   1.1 Шукаємо співпадіння в колонках name, references, description.
   1.2 Якщо не знайдено повного співпадіння і артикул містить крапку "." - то шукаємо по значеню артикула до крапки. Наприклад було "AZ10331.41" - якщо по ньому не знайшли нічого то шукаємо по "AZ10331".
   (Якщо в одній клітинці "Артикул" кілька значень через кому/крапку з комою - перевіряються всі.)

2. Для кожного джерела окремо додаємо колонку з найдешевшою знайденою ціною:
   - "Ціна Рамос"     - ціна з tvk_ramos_prices.xlsx;
   - "Ціна Агродоктор" - ціна з agrodoctor_prices.xlsx.
   Якщо в конкретному джерелі товар не знайдено - ціна 0.

3. Колір конкретної клітинки з ціною:
   - зелений #82d200 - якщо ціна в цьому джерелі нижча ніж у нас ("Ціна Агродрузі") і товар є в наявності;
   - червоний #ff6f6d - якщо товару немає в наявності, або ціна дорівнює/вища за нашу, або товар не знайдено (ціна 0).

4. Якщо ціна знайдена (відмінна від нуля) - вона стає клікабельним гіперпосиланням, що веде на сторінку товару у конкурента.
