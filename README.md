Проєкт призначений для аналізу цін між магазинами.

Нам поступає товар - див файл Input.xlsx

З ним треба порівнювати ціни у конкурентів - це два файли:
agrodoctor_prices.xlsx
tvk_ramos_prices.xlsx

Утиліта яка порівнює ціни - check_prices_urls.py

Вона шукає чи в конкурентів є нижча ціна і записує результат у новий файл
Input_analyzed_with_urls_YYYY-MM-DD.xlsx (до назви дописується дата аналізу,
наприклад Input_analyzed_with_urls_2026-07-20.xlsx).

Запуск:
    pip install -r requirements.txt
    python check_prices_urls.py

Логіка за якою працює утиліта:

1. Перевіряєм для кожного рядка з файла Input.xlsx за значенням з колонки "Артикул" співпадіння в таблицях з цінами конкурентів.
   Важливі два моменти:
   1.1 Шукаємо співпадіння в колонках name, references, description.
   1.2 Якщо не знайдено повного співпадіння і артикул містить крапку "." - то шукаємо по значеню артикула до крапки. Наприклад було "AZ10331.41" - якщо по ньому не знайшли нічого то шукаємо по "AZ10331".
   (Якщо в одній клітинці "Артикул" кілька значень через кому/крапку з комою - перевіряються всі.)

2. Якщо товар знайдено і найнижча ціна конкурентів (колонка "price") нижча ніж у нас (колонка "Ціна Агродрузі") - в колонку "Рекомендація" нової таблиці пищемо цю ціну, в "Наявність" - значення з "availability", а в "Джерело" - назву конкурента ("agrodoctor" або "tvk-ramos") і в колонку "Посилання" - посилання на товар.
   2.1 Якщо в колонці "Наявність" значення "Немає в наявності" - колір голубий #80bcd7
   2.2 Якщо в колонці "Наявність" значення відмінне від "Немає в наявності" - колір зелений #82d200

3. Якщо товар знайдено, але найнижча ціна дорівнює або вище нашої - в колонку "Рекомендація" пишемо "Ціна оптимальна". Колір жовтий #fffc00

4. Якщо товар не знайдено - в колонку "Рекомендація" пишемо "Не знайдено". Колір червоний #ff6f6d

Кольором позначаються лише клітинки в колонках "Рекомендація" та "Наявність".
